DNS SinkHole

A small DNS sinkhole that quietly keeps bad domains away.

Download

> Quick Start

  1. Download & run the latest release.
  2. Choose a DNS server in Configuration → Upstream Resolver.
  3. Add blocklists (Optional)
    Paste blocklist URLs and click +.
  4. Start the sinkhole
    Click Start Sinkhole — the status will show: [+] Status: Running
  5. Enable system DNS (Required)
    Click Set System DNS so Windows uses the sinkhole.
  6. Browse normally
    The Query Log will display Allowed and Blocked domains.
  7. To stop using the sinkhole
    Just click Stop Sinkhole — your original DNS settings are automatically restored.

> Features

  • UDP-only DNS sinkhole — simple, fast, and reliable.
  • Block unwanted domains using built-in or custom blocklists.
  • Real-time query logging with a clean terminal-styled UI.
  • Local caching for faster repeat lookups.
  • One-click option to make Windows use the sinkhole for DNS.
  • Privacy-focused — no telemetry or analytics.

> How It Works

The app runs a local DNS server on 127.0.0.1:53. When a domain is requested:

> A DNS request comes in…
  ↓
> Check if the site is in the blocklist
   ├─ YES → Block it
   │        → Reply with a safe “blocked” address [0.0.0.0]
   │        → Status: BLOCKED ✓
   │
   └─ NO  → Continue…
             ↓
> Check the cache
   ├─ Cached → Return the cached answer ✓
   │          → Status: ALLOWED (cached)
   │
   └─ Not cached →
           Ask your normal DNS server for the answer…
             ├─ Reply received → Cache it → Return it ✓
             │                  → Status: ALLOWED
             │
             └─ No reply / timeout →
                     Return a safe error response ✓
                     (prevents browser “waiting for DNS…” hangs)

> Query complete.

> Requirements

  • Windows 10 / 11.
  • Administrator rights (for DNS redirection).

> Safety & Privacy

  • No analytics or user tracking.
  • DNS traffic remains local except when forwarding allowed queries.
  • EDNS Client Subnet (ECS) is not used.
